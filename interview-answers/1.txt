■ Q：オブジェクト指向のメリットは何ですか？

オブジェクト指向のメリットは、保守性・再利用性・拡張性が高いことです。
データと処理をクラスとしてまとめることで、変更の影響範囲を限定できます。

$user->getGreeting() のように、public メソッドを通して操作しているため、
内部のデータや処理に直接触れません。
その結果、クラス内部の実装を変更しても、外部のコードに影響しにくくなります。

class User {
    private $name;

    public function __construct($name) {
        $this->name = $name;
    }

    public function getGreeting() {
        return "Hello, " . $this->name;
    }
}

$user = new User("Naoki");
echo $user->getGreeting();
