# GitHub Traffic（アクセス解析）

GitHubにはリポジトリのアクセス状況を確認できる機能がある。

## 見る場所

1. Repositoryを開く
2. Insights
3. Traffic

## 指標

### Views
ページが開かれた回数

例  
同じ人が3回ページを見る  
→ Views = 3

### Visitors
ユニーク訪問者

例  
同じ人が10回ページを見る  
→ Visitors = 1

## 注意

- 過去14日しか見れない
- Botのアクセスが含まれる
- 正確な人数ではない

## 補足

PRや他のリポジトリのアクセスは含まれない。  
このリポジトリ内のページのみ対象。
